Fonts Directory
===============

Place custom TTF font files here if needed.

Recommended fonts:
  Roboto-Regular.ttf    — Body text (Apache 2.0 license)
  Roboto-Bold.ttf       — Headings
  RobotoMono-Regular.ttf — Timecodes

Download from Google Fonts: https://fonts.google.com/specimen/Roboto

Kivy will use its bundled fonts if no custom fonts are configured.
To use a custom font in KV language:

    Label:
        font_name: 'assets/fonts/Roboto-Regular.ttf'
